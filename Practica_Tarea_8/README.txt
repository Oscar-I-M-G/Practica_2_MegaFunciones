Tarea 8

Memoria ROM

Implementar una memoria ROM de 16 posiciones y 8 bits
de datos.
Utilizar para ello la megafunción LPM_ROM.
Los valores de cada posición de la memoria se especifican
con un archivo .mif. El contenido de cada posición es la
dirección multiplicada por 4. Utilizar la ayuda de la
megafunción para la edición del archivo .mif.
La dirección se introducirá con cuatro interruptores.
La salida de datos debe mostrarse en dos diplays de 7
segmentos, utilizando para ello el código hex_7seg.vhd
proporcionado.
